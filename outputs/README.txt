Rendered images (binary PPM, P6) go in this folder.

Notes for beginners:
- Files produced by the renderer use the "P6" binary PPM format. Each file is raw RGB bytes after a small ASCII header.
- Many image viewers support PPM; if yours doesn't, convert to PNG with ImageMagick or use an online converter.

Example commands

Windows (PowerShell):

  .\raytracer.exe inputs\example1.txt outputs\example1.ppm
  magick convert outputs\example1.ppm outputs\example1.png   # requires ImageMagick

macOS / Linux:

  ./raytracer inputs/example1.txt outputs\example1.ppm
  magick convert outputs\example1.ppm outputs\example1.png

Tips
- If the image looks too dark or bright, you may want to apply gamma correction when viewing or convert with a gamma-aware tool.
- The files `outputs/example1.ppm`, `outputs/example2.ppm`, and `outputs/example3.ppm` are generated by the `Makefile` targets `make example1`, `make example2`, and `make example3` respectively.
